config 级别

Git配置文件有三个级别，优先级如下：仓库级（local）> 全局级（global）> 系统级（system）

配置级别 位置 配置文件路径
仓库级（local） git 工作区间 D:\myGitRepo\.git\config
全局级（global） C 盘 user 路径 C:\Users\username\.gitconfig
系统级（system） git 安装目录 C:\Program Files\Git\etc\gitconfig
